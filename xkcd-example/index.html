XKCD's Frequency in HTML, CSS and JavaScript.

Adapted from Daniel Hough's original recreation of the XKCD cartoon, using the open-source JavaScript library eventedloop, and the data collected by the wonderful Jeremy Singer-Vine, it was easy to recreate Frequency, one of Randall Munroe's XKCD comics. The font is from here.
